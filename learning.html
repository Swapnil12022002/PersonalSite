This is a schematic representation of the list of balon d'or winners from 2017 to 2021 in a tabular form.
The table command is used to open to create a table. The thead command is used to mention the name of the categories in the table.
And the tr command is used to create a row in the table. Then the th command is used to give data in the cells. The th command is used only in case of thead commands.
The td command replaces the th command in the tbody section.
